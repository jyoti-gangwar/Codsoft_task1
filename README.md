# To-Do List Application
## Description
A GUI-based To-Do List application developed using Python and Tkinter.

## Features
- Add Tasks
- View Tasks
- Mark Tasks as Completed
- Delete Tasks
- Save Tasks using JSON

## Technologies Used
- Python
- Tkinter
- JSON
